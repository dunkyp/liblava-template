# liblava-template
Project template for <a href="https://git.io/liblava">liblava</a>

1. Clone
2. Rename
3. Build

```
$ git clone https://github.com/liblava/liblava-template.git

$ mv liblava-template project
$ cd project

$ git submodule update --init --recursive

$ mkdir build
$ cd build

$ cmake ..
$ make
```
